Bleeding-Edge

You can also retrieve the source files from our Git repository and create a bleeding-edge build by typing:

git clone {{ site.data.project.source_repository_mirror }} cd incubator-systemml mvn clean package

Using Maven

We strongly encourage our users to configure a Maven repository mirror closer to their location. Please read How to Use Mirrors for Repositories.

Dependencies

Add the following dependencies to your Maven pom.xml.

        <dependency>
        <groupId>org.apache.systemml</groupId>
        <artifactId>systemml</artifactId>
        <version>0.10.0-incubating</version>
        </dependency>
